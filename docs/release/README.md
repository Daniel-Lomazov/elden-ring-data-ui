# Release Management

This directory is the release-control surface for `elden_ring_data_ui`.

Current policy:
- Treat the project as `0.x` until a release decision explicitly justifies `1.0.0`.
- Keep release artifacts live and updated as code lands.
- Record blockers instead of assuming readiness.

- [Release checklist](./release-checklist.md) — steps required before each release
- [Changelog](./changelog.md) — user-facing change history
- [Release notes draft](./release-notes-draft.md) — in-progress release notes
- [Final release memo](./final-release-memo.md) — last completed release summary
